# Hacker-Terminal-Simulator
这是一款基于Web的赛博朋克风黑客终端模拟游戏。游戏呈现了一个沉浸式的虚拟入侵场景：绿色字符代码雨、CRT复古扫描线、霓虹终端窗口，以及多层级的黑客小游戏。玩家将以入侵者身份，从虚拟主机系统中开始渗透，通过模拟Terminal的方式输入指令进行探索、破解与反制。游戏包含多种交互内容：可输入命令的Terminal、真实的命令解释器ls/cd/cat…、指令自动补全Tab、命令历史记录、随机文件系统关卡、密码破解游戏、文本线索解密任务、实时"追踪反制"小游戏。所有内容均使用HTML+CSS+JavaScript/React实现。界面视觉灵感来自《黑客帝国》等作品，并加入个性签名字符流："Made By Haoze Zheng"作为特殊字符穿插在代码雨之中。这是我系统学习完Linux后的小成果。
